# Smart Docklands Bike Sharing Management System

This project was developed for the **Data Structures & Algorithms** module.

The application is based on the Mobility Innovation theme from the Smart Docklands Pilot Call. It is a Java application built using NetBeans with a Swing GUI that allows users to manage a shared bike system.

Features
- Add, view, update, and delete bikes 
- Track repair requests for damaged bikes
- Store objects using data structures
- Demonstrates inheritance, interfaces, and abstract data types

Data Structures Used
- Singly Linked List
- Queue
- ArrayList

 Technologies
- Java
- Java Swing
